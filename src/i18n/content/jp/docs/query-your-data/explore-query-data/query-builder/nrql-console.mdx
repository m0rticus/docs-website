NRQL コンソールは、実行している内容のコンテキストを維持しながら、 **[one.newrelic.com](https://one.newrelic.com/all-capabilities)**内のどこにでもデータをクエリするための簡単な方法です。サービスの監視、問題のトラブルシューティング、または単にプラットフォームの閲覧を行っている場合でも、NRQL コンソールを使用すると、現在のビューを離れることなく、クリックするだけでデータを探索して、表示されている内容をさらに理解できます。New Relic 内の任意の画面の下部バーをクリックするだけで、コンソールを使用してデータのクエリを開始できます。

NRQL コンソールを有効または無効にするには、 [ユーザー メニュー](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) に移動し、NRQL コンソール トグルを探します。
