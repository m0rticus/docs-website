NRQL 콘솔은 현재 수행 중인 작업에 대한 컨텍스트를 유지하면서 **[one.newrelic.com](https://one.newrelic.com/all-capabilities)** 어디에서나 데이터를 쿼리할 수 있는 빠른 방법입니다. 서비스를 모니터링하든, 문제를 해결하든, 아니면 플랫폼을 검색하든 상관없이 NRQL 콘솔을 사용하면 현재 보기를 벗어나지 않고도 단 한 번의 클릭으로 데이터를 탐색하여 현재 보고 있는 내용에 대해 더 자세히 이해할 수 있습니다. 콘솔을 사용하고 데이터 쿼리를 시작하려면 New Relic 내 화면의 하단 표시줄을 클릭하기만 하면 됩니다!

NRQL 콘솔을 활성화 또는 비활성화하려면 [사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) 로 이동하여 NRQL 콘솔 토글을 찾으십시오.
